# DevFolio - Personal Portfolio Template

[![Live Demo](https://img.shields.io/badge/demo-online-brightgreen.svg)](https://dev-folio-coral.vercel.app/)

DevFolio is a modern, clean, and responsive personal portfolio website template built with **Bootstrap 5**. It's designed for developers, designers, and creative professionals to showcase their work and skills effectively.

## 🚀 Live Demo
Check out the live version here: [https://dev-folio-coral.vercel.app/](https://dev-folio-coral.vercel.app/)

## ✨ Key Features
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Hero Section**: Includes a dynamic typing effect to greet visitors.
- **About Section**: Professional summary with interactive skill bars.
- **Services**: Highlights your core expertise (Web Design, Development, Photography, etc.).
- **Portfolio Gallery**: A sleek showcase for your latest projects.
- **Blog Section**: Share your thoughts and updates with a clean card layout.
- **Contact Form**: Integrated contact section for easy communication.

## 🛠️ Built With
- **HTML5** & **CSS3**
- **JavaScript** (ES6+)
- **Bootstrap 5** - For layout and components.
- **FontAwesome** - For high-quality icons.

## 💻 Getting Started
To run this project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mayarmohamed123/DevFolio.git
   ```
2. **Open the project**:
   Navigate to the project folder and open `index.html` in your favorite web browser.
